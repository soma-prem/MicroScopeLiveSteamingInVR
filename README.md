VirtualScope is a VR application that allows users to view live streams from a traditional microscope in a virtual reality environment. It integrates AI-based image analysis, making it ideal for education, research, and interactive science experiences.

Features

Live Microscope Streaming: Stream real-time images from your microscope to a VR environment.

AI Image Analysis: Integrated AI analyzes captured images for detailed insights.

Virtual Chemistry Lab: Interact with a virtual lab setup while viewing live microscope feed.

Multiple Scenes:

Connection Scene: Input IP address and port number for your microscope feed.

Display Scene: Live streaming screen, AI interaction panel, and virtual lab.

Image Storage: All images can be stored in Azure Blob Storage for later review.

VR Compatibility: Designed for immersive VR experiences
